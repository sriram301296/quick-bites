Goal

Write a program that, given a Dictionary of words in English, finds the longest word such that all prefix-substrings are also valid words.

Contract

Input: Set dictionary Output: String longestWord, suchThat for 0 <= n <= len(longestWord)-1, longestWord.substr(0,n) is a valid word in dictionary

Examples

Dictionary

a bi cab i mobi nab obi zap

Output

mobi, since (mobi, obi, bi, i) are all valid words according to the dictionary

Notes

  • Empty space is not a valid word
  • If there are more than 1 such words, print any
  • For reference dictionary file: https://github.com/dwyl/english-words/blob/master/words_alpha.txt
  • The output for the above dictionary file will be "suprising" (though it's not correct spelling, the dictionary file has it).
